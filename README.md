##動畫設計流程

* 問題分析：想訴說什麼故事、需要哪些物件？要發生哪些動作？

* 構思：編寫故事腳本，即故事板繪製 story board。

* 實現：依序製作場景，並將腳本轉製變成 flash 動畫

* 測試：修正和除錯(debug)。
